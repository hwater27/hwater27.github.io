4. 단어의 의미

단어란 무엇일까?

  1. [표준국어대사전] 분리하여 자립적으로 쓸 수 있는 말이나 이에 준하는 말

  2. 문장을 구성하는 단위

    • 문장에서 분리될 수 있는 단위 (소리 연쇄)
    • 의미를 알 수 있는 단위

  3. 한 단어가 담고 있는 정보는 언어마다 달라 보인다.

    • Engenni (Thomas 1978)

      A     ta   na wa   omu.
      one go to seek house
      ⇒ Let’s go look for the house.

    • Mohawk

      Teninonhsihsakha.
      te-ni-nonhs-ihsak-ha.
      you.and.I-two-house-seek-go.and
      ⇒ Let’s go look for the house.

    ⇒ Engenni어 문장 = Mohawk어 단어?

  4. 그럼에도 단어를 안다는 것은?

    • 단어가 몇 개인지 아는 것
    • 단어의 뜻을 아는 것
    • 말하고자 하는 바를 어떻게 부르는지 아는 것

  5. 단어를 어떻게 알아차릴 수 있을까?

    • 머릿속 사전(심성 어휘부, mental lexicon)의 정보

  6. 심성 어휘부에 담긴 정보 ver. 1

    • 단어의 발음, 의미
    • 문장에서 담당할 수 있는 역할
    • 다른 단어와의 관계(유의어, 반의어 등)

4.1. 단어의 의미 정의

  1. 언어 표현의 ‘지시 reference’

    • 한 언어 표현의 의미는 그 표현이 세상에서 가리키는 대상들의 집합
      • (e.g.) 고양이 ← 아메리칸 쇼트 케어 고양이, 페르시안 고양이, 맹크스 고양이, ⋯
      • (e.g.) 자동사 swim (수영하다) ← 수영을 하는 사람들의 집합

  2. 언어 표현의 ‘개념 sense’

    • 한 언어 표현의 의미는 표현을 알고 있는 사람이 떠올리는 심리적 표상
      1. 사전식 dictionary-style
        : 단어를 설명하는 또다른 단어의 의미를 통해 파악
        • (e.g.) 고양이
          … 원래 아프리카의 리비아살쾡이를 길들인 것으로, 턱과 송곳니가 특히 발달해서 육식을 주로 한다 …
      2. 심적 영상 mental image
        : 머릿속에 떠오르는 영상

  3. 지시 reference만으로 언어의 의미를 모두 포함하기는 어려움

    • 구체적인 지시물이 없는 언어 표현도 있음
      • (e.g.) 사랑, 증오, 충성, 맹세, 추억 - 추상 명사
      • (e.g.) 매우, 많이, 늘, 자주 - 부사
    • 동일한 지시물을 가리키는 여러 언어 표현이 존재함
      • (e.g.) 금성, 샛별, 개밥바라기

  4. 개념 concept만으로 언어의 의미를 모두 포함하기는 어려움

    • 객관적으로 파악하기 어려움
    • 개인마다 언어 표현에 대해 떠오르는 바가 다를 수 있음

⇒ (개념이 밀접하게 관계되는) 단어들 간의 관계(공통점과 차이점) 등을 통해 단어의 의미를 파악하고자 함

  • → 구조주의적 어휘 의미론
  • → 장 이론과 성분 분석

개념에 기반을 둔 논의: Jakendoff의 ‘개념문법‘
“개념 문법은 어떻게 개념원소가 모여 단어의 의미를 이룰 것인지 나타내야 한다"
"우리는 단어가 엄격하고 정의를 갖고 있다는 생각을 포기해야 한다 ⋯ 적절한 상황적 민감성을 갖춘 정의를 만드는 것도 가능하다”

(cf) 생성 언어학은 ‘지시’에 기반을 두고 진리조건적 의미론을 발전시킴

4.2. 단어의 의미 분석 방식: 장이론과 성분 분석

  • 연속적인 색깔 사진은 ‘오색 무지개’인가? 아니면 ‘일곱 색깔 무지개’인가?
    • 사람들의 눈에 비친 무지개 색깔은 다르지 않음
    • 어휘와 어휘에 따른 구획의 차이
      ⇒ 개별 어휘 항목의 의미는 일정한 의미 영역 안에서 다른 어휘 항목과의 관계에 의해 규정된다.

4.2.1. 장이론

  1. 의미장: 밀접한 관계에 있는 언어 표현들의 집합

    • (e.g.) 성적의 등급 - 매우 잘함, 잘함, 보통, 노력 요함
    • (e.g.) 색채 - 하양, 검정, 빨강, 파랑, 노랑
    • (e.g.) 친족 - 할아버지, 할머니, 아버지, 어머니, 언니

  2. 언어 전체를 반영하는 의미로부터 세분화되어 가면서 작은 의미장들이 존재

    • (e.g.) ‘길짐승’의 의미장은 ‘소, 말, 개’의 상위장인 동시에 ‘날짐승, 인간’ 등과 함께 ‘동물’의 부분 장을 구성함

  3. 언어의 의미는 같은 장 속에 있는 다른 표현들과의 상호 관계 속에서 결정됨

    • (e.g.) ‘매우 잘함-잘함-보통-노력 요함’에서의 ‘잘함’의 의미
      • 4단계로 나뉘어 있음을 알아야 함
      • ‘잘함’ 아래에 ‘보통, 노력 요함’, 위에는 ‘매우 잘함’이 있음을 알아야 함
      • ‘잘함-노력 요함’에서의 ‘잘함’과 차이가 있음

  4. 의미장은 언어마다 다르다.

  • 레오 바이스게르버 Leo Wisgerber의 <언어적 중간 세계>
    “어휘는 책상에 대한 중간 세계에 대한 사고상”

  • 장이론의 초기 목표: 언어마다 고유한 세계관 밝히기

  1. 어휘를 묶고 가르는 언어적 근거

    착용물: 옷, 신발, 모자, 양복, 구두, 중절모

    • 그는 회의에 참석할 때면 항상 양복을 입는다.
    • 그는 양복을 입을 때면 항상 구두를 신는다.
    • 그는 코트를 입을 때면 항상 중절모를 쓴다.

    어휘 의미의 존재 양상: 계열 관계와 결합 관계를 통해 설명 가능

    • 계열 관계: 교체할 수 있는 말들 간의 관계
    • 결합 관계: 결합할 수 있는 말들 간의 관계

  2. 의미장의 분절 원리

    • 의미장은 일정한 기준에 따라 개념적으로 구획되어 있음
      • 구획의 기준을 파악함으로써 어휘들의 개별적 특성을 파악할 수 있음
      • 다른 어휘소의 결합 관계를 통해서 의미장의 구성이 정교해질 수 있음
    • (e.g.) 덥다, 뜨겁다, 따뜻하다, 뜨듯하다

      • 공통점: 온도

        1. 높은 온도의 의미장: 덥다, 뜨겁다
        2. 낮은 온도의 의미장: 서늘하다, 미지근하다

      • 주어와의 호응 관계를 고려한 세분화

        • 방 안이 춥다/덥다
        • 방바닥이 차갑다/뜨겁다
        • 방 안이 서늘하다/따뜻하다
        • 방바닥이 미지근하다/뜨듯하다
        1. 비접촉 감각의 의미장: 덥다, 따뜻하다
        2. 접촉 감각의 의미장: 미지근하다, 뜨듯하다

4.2.2. 의미 성분 인식과 성분 분석

  1. 의미 성분 인식

    • 의미장 내 어휘들의 내부 관계 분석 과정에서 의미 성분이 추출됨

  2. 성분 분석
    : 독립적인 이론이 아닌, 의미를 분석하고 기술하는 방법을 말함

    • 한 단어의 의미는 더 작은 의미로 구성됨
    • 의미 단위는 의미상의 공통점과 차이점에 근거하여 설정
    • 정보의 있고 없음을 표시하는 +, - 기호를 사용하여, 더 적은 단위로 표현 가능

  3. 성분 분석의 예

    • 구시렁거렸다, 중얼거렸다, 웅얼거렸다, 속닥거렸다
      → 포괄하는 의미: 잘 들리지 않는 목소리로 계속 말하다.
      → 공통 의미 성분: [말하다], [잘 들리지 않음], [반복적임]

      A. 분류 1 [불만]: 구시렁거리다 / 중얼거리다, 웅얼거리다, 속닥거리다
      B. 분류 2 [혼잣말]: 구시렁거리다, 중얼거리다, 웅얼거리다 / 속닥거리다
      C. 분류 3 [입속말]: 구시렁거리다, 중얼거리다, 속닥거리다 / 웅얼거리다

      구시렁거리다중얼거리다웅얼거리다속닥거리다
      [불만]+---
      [혼잣말]+++-
      [입속말]--+-

  4. 성분 분석의 장점

    • 의미 관계를 명확히 파악할 수 있음
      • 분류 체계가 잘 짜인 어휘 분석에 적용: 조리어, 친족 관계명 등
    • 의미적 호응 양상을 설명할 수 있음
      • (e.g.) 모자를 쓰다(입다, 신다, 끼다 X)
        모자가 ’+[얼굴부위]‘를 성분으로 가졌다고 하면, 모자와 일치하는 성분을 가진 ‘쓰다’를 선택하는 것으로 설명할 수 있음

  5. 한계

    • 일반 언어에서 사용되는 단어를 분석에 그대로 사용함
    • 보편적인 의미 성분 추출이 쉽지 않음
      • 의미 성분의 타당성에 대한 근거 부족
      • 어느 정도의 의미 성분 목록이 필요한지 분명하게 제시되지 않음
    • 어휘의 의미가 어떻게 합성되어 문장의 의미를 이루는지는 설명되지 않음

4.3. 의미관계

계열 관계: 같은 범주의 어휘 사이에 맺어지는 관계. 유의 관계, 반의 관계, 상하 관계, 부분-전체 관계가 있다.

4.3.1. 유의 관계

  1. 정의

    • 비슷한 의미를 가지는 서로 다른 단어들의 쌍
      • (e.g.) 사람-인간, 손목-팔목, 경험-체험, 흠집-기스-스크래치

  2. 유의어 생성 이유

    • 관계, 세대, 상황, 격식 등에 따른 선택
      • (e.g.) 나이-연세, 승인/허락-컨펌

  3. 유의어의 의미 차이를 검증하는 방법

    1. 교체 검증법: 동일한 문맥을 제시하고 유의 관계를 이루는 단어를 바꾸어서 그 문장이 성립하는지를 확인하는 방법

      • {씨/씨앗}을/를 {뿌리다/심다}

      • 참외 {씨/*씨앗}을/를 {뱉다/발라내다}

        → ‘씨앗’: 농사와 관련된 경우에 제한적으로 쓰임

    2. 반의어 검증법: 유의어를 이루는 두 단어의 반의어를 대응시켜서 의미의 차이를 발견하는 방법
      • ‘작다’-크다: 크기 / ‘적다’-많다: 수량
      • 그러나 두 단어의 반의어가 동일한 경우, 적용하기 어려움
        • (e.g.) 아래 vs 밑 → 반의어: 위
          • 교체 검증법을 보조적으로 사용 (자세한 사항은 {아래와/*밑과} 같다.)
    3. 복합어 검증법: 복합어 결합 양상 비교
      • (e.g.) 허공 vs 공중

        • 허공이 포함된 복합어: 허공, 허공계

        • 공중이 포함된 복합어: 공중 그네, 공중 누각, 공중 분해, 공중비행, 공중회전

    • 절대적인 검증법은 없음

    • 왜 이러한 차이가 나타나는지에 대한 이유 설명이 중요함

4.3.2. 반의 관계

정의

  • 어휘의 의미가 서로 반대되거나 대립되는 의미 관계를 이루는 단어들의 쌍
  • 대부분 같은 의미를 갖되 하나의 기준이거나 의미 특성의 차이로 대립되는 관계

공유하는 특성에 따른 분류

  1. 상보 반의어

    • 두 어휘가 특정 의미 영역을 상호 배타적으로 양분하는 경우
      • (e.g.) 홀수, 짝수
    • 특징1: 두 단어를 동시에 긍정하거나 부정할 수 없음
      • *이 자연수는 홀수이기도 하고, 짝수이기도 하다.
      • *이 자연수는 홀수도 아니고, 짝수도 아니다.
    • 특징2: 정도 부사의 수식을 받지 않으며, 비교 표현도 쓸 수 없음
      • *영희는 대학에 {약간/조금/엄청} {합격했다/불합격했다}.
      • *영희는 철수보다 대학에 더 {합격했다/불합격했다}.

  2. 정도 반의어

    • 저울의 양 끝처럼 대립하는 두 단어 쌍 사이에 중간 지대가 있는 반의 관계
      • (e.g.) 길다/짧다, 가볍다/무겁다, 작다/크다
    • 특징1: 두 단어를 동시에 긍정하거나 부정할 수 있음
      • (e.g.) 내 친구는 키가 크지도 않고 작지도 않아.
    • 특징2: 정도 부사의 수식을 받으며, 비교 표현도 쓸 수 있음
      • (e.g.) 방이 엄청 넓다.
      • (e.g.) 친구 방이 내 방보다 넓다.

  3. 관계 반의어(상호 반의어)

    • 두 단어 중에서 하나가 성립하면 반대편의 단어도 반드시 존재하는 상호적 관계를 가짐
      • (e.g.) 사다/팔다, 주다/받다, 때리다/맞다

  4. 방향 반의어

    • 어떤 사건이나 행위의 과정이나 방향이 반대될 때 생기는 반의 관계
      • (e.g.) 오다/가다, 열다/닫다
    • 관계 반의어와 차이점: 하나의 행위가 있다고 해서 반드시 반대 방향의 행위가 있지는 않음
      • 문을 열었다고 해서 반드시 닫는 행위가 있어야 하는 것은 아님

4.3.3. 상하위 관계

  1. 정의

    • 어휘들 사이에 상하 구조를 이루면서 서로 포함의 관계가 성립하는 경우
      • 해당 언어에서 세계를 분류하는 방식과 연관됨
    • (e.g.) 새 - 독수리, 참새, 솔개, 올빼미, 제비, 비둘기, 딱따구리, 기러기, ⋯

  2. 하위어가 참이면 상위어도 참이지만, 상위어가 참이라고 해서 하위어가 참은 아니다.

    • (e.g.) 독수리가 하늘을 날아간다. (참) → 새가 하늘을 날아간다. (참)
    • (e.g.) 새가 하늘을 날아간다. (참) → 독수리가 하늘을 날아간다. (참?)

4.3.4. 부분-전체 관계

  1. 정의

    • 한 단어가 다른 단어의 부분이 되는 관계
    • 시공간에 함께 존재하는 일련의 대상들을 조립하는 것
      • (e.g.) 전체어 ‘자동차’ | 부분어: 엔진, 바퀴, 핸들, 브레이크, 의자

  2. 특징1: 부분어가 참이더라도, 전체어가 참이 아닐 수 있음

    • (e.g.) 다리가 아프다. (참) → 몸이 아프다. (참)
    • (e.g.) 타이어를 샀다. (참) → 차를 샀다. (참?)

  3. 특징2: 부분어인 한 단어가 여러 개의 다른 전체어와 연관될 수 있음

    • (e.g.) 손잡이 - 전체어: 문, 버스, 망치, 컵

단어의 의미 관계

  • 기준에 따라 여러 관계로 정의될 수 있음

  • 관건: 기준에 대한 근거, 기준의 설명력

  • 지시하고 표현해야 할 대상이 변화하고 증가함에 따라 기존의 단어가 표상할 수 있는 의미가 확대되기도 함

    • 단어의 다의성
      • 중심 의미: 가장 기본적이고 핵심적인 의미
      • 주변 의미: 문맥이나 상황에 따라 그 범위가 확장되어 다른 의미를 갖게 됨
      • (e.g.) 손을 깨끗이 씻다. / 손이 모자라도(노동력), 손이 크다(씀씀이)
    • 의미적 신어 (송현주 2022)
      • 분위기 깡패, 실물 깡패
      • 팩트 폭행
        → 은유: 의미 생성의 기제?

4.4. 단어의 의미 생성 기제로서의 은유

  1. 정의: 한 종류의 사물을 다른 종류의 사물을 이용하여 이해하는 방식

    • 일상 의사소통 과정에서 널리 사용되는 표현 방식
    • 단어의 다의성을 생성하는 주요 기제

  2. 기능

    • 구체화, 간결화, 선명화
      • 삶은 미로의 연속이다. → 추상적 설명 대상을 구체화
      • 내 컴퓨터가 갑자기 말을 안 듣는다. → 복잡한 설명 대상을 간결화
      • 드디어 선거전이 시작되었다. → 설명 대상을 생생하게 전달

  3. 은유 과정

    • (e.g.) 시간이 정말 빠르게 흘렀다.
      1. 서로 다른 영역에 속하는 두 가지 개념이 관여됨: ‘시간’과 ‘액체’
      2. 두 개념이 유사성에 기초하여 교체되어 쓰이면서 은유 발생
        • 시간 [+추상성][+유동성]
        • 액체 [+구체성][+유동성]
      3. 익숙하고 구체적인 개념 영역(근원 영역)의 개념이 상대적으로 낯설고 추상적인 개념 영역(목표 영역)으로 대응됨
        • (e.g.) 시간을 아껴 써야 한다.
          • 형식화: 시간은 돈이다. *목표영역: 시간 | 근원영역: 돈
        • (e.g.) 깨지지 않을 사랑을 하자.
          • 형식화: 사랑은 그릇이다.
            • 목표영역: 사랑 | 근원영역: 그릇
          • 추상적이고 모호한 개념을 구체적 실체로 간주
          • 근원영역의 직접적이고 구체적 속성을 이용하여 추상적 목표 영역을 효과적으로 이해
        • (e.g.) 이번 승리로 사기가 올랐다/사기를 떨어뜨리는 행동을 하지 맙시다.
          • 형식화: 행복은 위이고, 불행은 아래이다.
            • 목표영역: 행복 | 근원영역: 위
            • 목표영역: 불행 | 근원영역: 아래
          • 일상생활에서의 물리적 경험이 반영되어 새로운 은유 체계를 형성함
